# Monocourse Python 项目

这是一个初始的 Python 项目骨架，包含基础目录结构、依赖管理文件，以及一个简单的示例脚本。

## 目录结构

```
Monocourse/
├── monocourse/
│   ├── __init__.py
│   └── main.py
├── requirements.txt
└── README.md
```

## 快速开始

1. 创建虚拟环境并激活（可选推荐）：

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
```

2. 安装依赖：

```bash
pip install -r requirements.txt
```

3. 运行示例脚本：

```bash
python -m monocourse.main
```

## 后续

如需添加第三方依赖，请在 `requirements.txt` 中列出并重新安装。
